## Project Overview

ShopMitra Pro compares products from multiple e-commerce platforms like Flipkart, Amazon, and more. It showcases professional-grade product images (similar to official product photography) and highlights the best available prices. The interface is responsive, clean, and user-friendly.

---

## Features

- Real professional product images with fallback support  
- Multi-platform price comparison with best deal highlighting  
- Responsive layout compatible with desktop and mobile  
- Lazy loading for images to enhance performance  
- User search functionality for filtering products
